This script makes Output images for build cards out of the Common ASS(Armor Set searcher) from the wiki.db site.
How to Run the Tampermonkey Script.

Step 1. Install Tampermonkey on your browser. https://www.tampermonkey.net/ from here.

Step 2. Add Script and Copy and Paste the Script from the Code File into it.

Step 3. Reload Page and enjoy.
